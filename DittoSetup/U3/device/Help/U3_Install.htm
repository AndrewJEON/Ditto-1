Thank you for installing Ditto for U3 Devices.

 

   What is Ditto?
 
Ditto is an extension to the Windows Clipboard.  You copy something to the Clipboard and Ditto takes what you copied and stores it in a database to retrieve at a later time.
   U3 Device Specific
 
Due to speed and space limitations Ditto's database of clips is stored on the host machine while Ditto is running and on the device in a zipped format when Ditto is not running.  The database is copied down from the U3 device on startup and copied back to the U3 device on exit.  If the device is yanked out of the USB drive your data will be lost.  It is verify important to eject the device properly.  While Ditto is running Ditto will copy its database to the device once an hour so all is not lost if the device is ejected improperly.  Ditto's database can be manually copied to the device by right click on the task tray icon and selecting 'copy database to device'.

Changes from standard Ditto

  • Settings are now stored in a .ini file on the device 
  • Database is stored on the device in a zipped format, copied down on startup and zipped back up on exit.
  • Name copy and Paste are not supported 
  • New method for tracking focused window
  • Defaulted maximum clips to 250

 

    Basic Usage
 
  • Run Ditto.
  • Copy things to the clipboard, e.g. using Ctrl-C with text selected in a text editor.
  • Open Ditto by clicking its icon in the system tray or by pressing its Hot Key which defaults to Ctrl + ` – i.e. hold down Ctrl and press the back-quote (tilde ~) key.
  • Double click or press enter on the item to paste it to the previous window.
